변수의 사용


변수의 사용 규칙
1. 대소문자를 구분
2. 영문, $, _ 로 시작해야 함
3. 영문, $, _ 를 포함하고 특수문자는 포함 할 수 없다.
4. 키워드나 예약어는 사용 할 수 없음
변수의 범위
1.전역 변수 : window객체에 포함되는 변수. 다른 함수들에서 공통으로 사용 할 수 있다. 값이 유지됩니다.
2. 지역변수 : 함수나 객체 내부에서 선언되어 실행이 종료되면 사라짐(var let const)
변수의 선언과 저장
1.var/let 키워드를 사용해서 변수에 저장.
2.const 키워드는 변수를 상수로 만든다.
3.javascript의 변수의 타입을 저장되는 값에 의해서 결정된다.
타입의 종류
  • 문자(string)
  • 숫자(number)
  • 논리형(boolean)
  • 널(Object/null)
  • 객체(Object/array)
  • 함수(function)
function의 호출
  • onload를 통한 익명함수의 자동
  • 이벤트 속성을 통한 onclick=함수
  • 이벤트 속성을 통한 호출 global함수 onclick = "javascript:alert('실행')"
  • addEventListner을 통한 function의 실행
